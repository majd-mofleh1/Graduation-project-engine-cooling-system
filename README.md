# Graduation-project-engine-cooling-system
Design and Thermal Analysis of an Improved Cooling System for a Four-Cylinder Gasoline Engine
Overview
This graduation project presents an improved engine cooling system based on a split-cooling strategy using an electric water pump and an electronically controlled thermostat.

Objectives
Improve engine thermal management.
Reduce temperature variations and hot spots.
Evaluate coolant flow under different operating conditions.
Tools
MATLAB
Fusion 360
Heat Transfer Analysis
Thermodynamics
Fluid Mechanics
